# vlatten

[![NPM version][npm-image]][npm-url]
[![Downloads][download-badge]][npm-url]

> I hate live coding exercises. Hopefully this makes up for it.

## Install

```sh
npm i -D vlatten
```

## Usage

```js
import vlatten from "vlatten"

vlatten([1, [2], [3, ['a']]]) // -> [1, 2, 3, 'a']
```
